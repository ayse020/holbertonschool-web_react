# Task 0 - Creating an interface for a student

## Description

This project is part of the Holberton School TypeScript curriculum.

The goal of this task is to create a Student interface using TypeScript, create two student objects, store them in an array, and render their information in an HTML table using Vanilla JavaScript.

## Learning Objectives

- Understand how to use TypeScript interfaces
- Create strongly typed objects
- Work with arrays of objects
- Manipulate the DOM using Vanilla JavaScript

## Requirements

- Create a Student interface with:
  - firstName (string)
  - lastName (string)
  - age (number)
  - location (string)
- Create two students: student1 and student2
- Store them in an array called studentsList
- Render a table using JavaScript
- Each row must show:
  - First name
  - Location
